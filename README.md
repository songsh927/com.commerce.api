# COMMERCE API SERVER  

### 1. 프로젝트 소개
  
커머스 API 서버 구현을 통해 *동시성문제*, *Docker*, *Docker Compose*, *Kafka*, *Redis* 등을 사용하여 해당 기술들이 어떤 환경에서 어떻게 사용해야 하는지 학습하기 위해 진행한 프로젝트입니다.  
인프런 강좌를 통해 진행했습니다.
